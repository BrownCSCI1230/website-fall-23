Markdown

Markdown is my note-taking language of choice.

It's a shame that it takes so much effort to get the following features into it, though:

1. LaTeX
2. Code blocks
3. Directives
   - e.g. "info" or "warning" boxes
4. Tables
5. Collapsible sections
6. Nicely-formatted images

Still, I'm pretty excited about [MDX JS](https://mdxjs.com)!
